Module 2 clinic_demo data dictionary
This file documents the small practice datasets used in Module 2.

[labs.csv]
patient_id      : Patient identifier (e.g., P001)
visit_date      : Visit date (YYYY-MM-DD)
test_name       : Lab test name (Glucose, HbA1c, etc.)
result_value    : Measured result value
unit            : Measurement unit (mg/dL, mmol/L, etc.)

[vitals.csv]
patient_id      : Patient identifier (e.g., P001)
visit_date      : Visit date (YYYY-MM-DD)
heart_rate      : Heart rate (beats per minute)
bp_systolic     : Systolic blood pressure (mmHg)
bp_diastolic    : Diastolic blood pressure (mmHg)
temperature_c   : Body temperature (Celsius)
resp_rate       : Respiratory rate (breaths per minute)

[meds.tsv]
patient_id      : Patient identifier (e.g., P001)
med_name        : Medication name
dose            : Dose value
unit            : Dose unit (mg, mcg, puffs)
frequency       : Dosing frequency (QD, BID, QHS, PRN)

Notes:
- These datasets are synthetic and for teaching only.
- Files are plain text so they can be previewed with head/tail/less.
- CSV files use commas; TSV files use tabs.
- In Module 2, students should save outputs in practice/.
