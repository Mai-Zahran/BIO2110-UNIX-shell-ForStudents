# Module 1 Practice Data (BIO2110)

This folder is generated by an instructor script.
Students: you do NOT need to run the generator.

What this module contains:
- ehr_demo/raw/      synthetic CSV tables for previewing and navigation practice
- ehr_demo/docs/     a data dictionary and short notes file
- ehr_demo/results/  empty output folder (you will create files here later)
- practice/          files for safe file operations (mkdir, touch, cp, mv, rm)

Beginner routine (repeat often):
1) pwd   (Where am I?)
2) ls -F (What is here?)
3) Move with cd, then re-check pwd and ls -F

Professional habit:
- Never edit “raw” data in place.
- Copy raw files into a workspace before changing anything.

AI-aware note:
- You may ask AI tools to explain errors or commands.
- You must verify everything by running commands yourself.
- Never paste private or sensitive data into AI tools.
