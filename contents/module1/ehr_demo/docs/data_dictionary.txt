Data dictionary (teaching demo) — Module 1 ehr_demo

Files are synthetic and designed for beginner-friendly shell practice.

File: patients.csv
- patient_id : P### synthetic patient identifier
- sex        : F or M (simplified for teaching)
- age_group  : categorical age band (20-29, 30-39, etc.)
- zip3       : first 3 digits of ZIP code (coarse location)

File: visits.csv
- visit_id   : V#### synthetic visit identifier
- patient_id : links to patients.csv patient_id
- visit_date : YYYY-MM-DD visit date
- visit_type : Outpatient, ED, Inpatient

Suggested beginner checks:
- Preview headers:
  head patients.csv
  head visits.csv

- Confirm row counts:
  wc -l patients.csv
  wc -l visits.csv

- Practice safe copying:
  cp ehr_demo/raw/patients.csv practice/patients_copy.csv

Reminder:
- These files are for learning. Do not treat them as real clinical data.
